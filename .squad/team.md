# EuroleagueClaw — Squad

## Project Context

**Project:** EuroleagueClaw — A TypeScript/Node.js bot that posts live EuroLeague basketball game updates, fantasy league info, and trivia to Telegram group chats.
**Stack:** TypeScript (strict, ESM), Node.js ≥22, grammy (Telegram), better-sqlite3, node-cron, Zod, Pino, vitest
**Architecture:** Hexagonal (Ports & Adapters) — ChatPort, StatsPort, FantasyPort, StoragePort, SchedulerPort
**User:** Filip Tanic
